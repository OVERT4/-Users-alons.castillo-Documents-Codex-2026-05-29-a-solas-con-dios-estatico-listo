Aviso de privacidad

A Solas con Dios

Este espacio existe para ayudarte a acercarte a Dios con honestidad, privacidad y sin juicios.

Qué datos se usan

La página puede guardar en este dispositivo tu nombre local, tus escritos, entradas del diario, versículos favoritos, preferencias de tema, hábito espiritual y recordatorios.

Dónde se guardan

Todo se guarda únicamente en el navegador de este dispositivo usando localStorage. No hay backend, cuentas en línea, base de datos ni servidor propio que reciba lo que escribes.

Quién puede verlo

Nadie más desde esta página. Tus textos no se envían para ser revisados por otra persona. Si alguien tiene acceso físico a tu dispositivo o a tu navegador desbloqueado, podría ver lo que quedó guardado ahí.

Servicios externos

El versículo del día puede solicitarse a una API bíblica gratuita y algunas imágenes se cargan desde Wikimedia Commons. Tus escritos personales no se mandan a esos servicios.

Cómo borrar tu información

Puedes borrar tus entradas una por una dentro de la página. También puedes limpiar los datos del sitio desde la configuración de tu navegador si quieres eliminar todo lo guardado en este dispositivo.
